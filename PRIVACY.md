# 개인정보처리방침

최종 수정일: 2025년 1월 1일

## 1. 수집하는 정보

GitLab Auto Reviewer는 다음 정보만 수집합니다:
- 사용자가 직접 입력한 GitLab 리뷰어 이름 또는 사용자명

## 2. 정보 저장 방식

- 모든 데이터는 사용자의 **브라우저 로컬 저장소**(Chrome Storage Sync)에만 저장됩니다
- **외부 서버로 전송되지 않습니다**
- 개발자는 사용자 데이터에 접근할 수 없습니다

## 3. 정보 사용 목적

수집된 정보는 오직 다음 목적으로만 사용됩니다:
- GitLab Merge Request 페이지에서 리뷰어를 자동으로 선택하는 기능 제공

## 4. 정보 공유

- 어떠한 제3자와도 데이터를 공유하지 않습니다
- 외부 서버로 데이터를 전송하지 않습니다
- 분석 도구(Google Analytics 등)를 사용하지 않습니다

## 5. 데이터 삭제

사용자는 언제든지 다음 방법으로 저장된 데이터를 삭제할 수 있습니다:
- 확장 프로그램 팝업에서 리뷰어를 개별 삭제
- 확장 프로그램 삭제 시 모든 데이터가 자동으로 삭제됩니다

## 6. 보안

모든 데이터는 브라우저의 기본 보안 메커니즘으로 보호됩니다.

## 7. 개인정보처리방침 변경

본 개인정보처리방침은 법령 또는 서비스의 변경사항을 반영하기 위해 수정될 수 있습니다.
